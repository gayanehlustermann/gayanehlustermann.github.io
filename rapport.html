Rapport projet page Web

HTML:

Grâce a HTML (Hyper Text Markup Language) il est possible de programmer un site web. HTML, est un document texte, qui utilise le markup language. Ce sont des tags diffrents avec différentes utilités. Par exemple le tags 'strong' qui permet de mettre en gras une lettre, mais ce tag n'est pas le plus important. Les plus importants sont les tags tel que: 'html', 'head' et 'body', c'est un peu construit comme un corps humain. Dans le header vont êtes écrites toutes les informations les plus importantes pour la page web.(Ici vous trouvez tout les tags pour la "Tete".)

CSS:

Grâce a CSS, il est possible d'embellir le site web écrit avec HTML. Il est possible de faire plusieurs choses différentes, par exemple on peut changer la couleur de font ou de l'écriture, on peut aussi changer le style de l'écriture. Il est possible de faire les mêmes choses avec HTML, mais avec CSS on peut faire un style pour toutes les pages, c'est très pratique!

Rapport projet créer une Carte

Base de données publique utilisée:

Pour ce projet il fallait créer une carte avec une base de données publique. J'ai trouvé mes données sur Portail des données ouvertes de l'UE. J'ai utlisé les tables cvs de 2011/2012 pour les étudiants et les professeurs en erasmus. Les informations que j'ai utlisé sont le pays de départ et le pays de destination.

Démarche du projet:

En premier, j'ai analysé les fichiers csv afin de savoir quelles informations j'ai besoin pour le projet. Après, j'ai écrit un programme Python qui peut lire les fichiers csv utilisés. Ensuite j'ai écrit une class, pour ramasser (dans des dicionnaires) toutes les informations nécessaire pour faire les graphiques dans les popups, c'est-à-dire les pays de départ et de destination, pour les étudiants et les professeurs. Le programme fait un grand dictionnaire avec tout les pays de L'EU (key), puis en value il y a une liste avec toutes les destinations à partir du pays en key.

Lors de ce projet j'ai rencontré plusieurs problèmes. Premièrement, c'était difficile de trouver un fichier de la bonne longueur et avec des informations qu'on peut représenter sur un graphique. Ensuite, dans le fichier csv que j'ai utlisé, les pays était en 'coutrycode'. La Belgique a plusieurs 'countriecodes', alors il fallait les combiner afin d'avoir qu'une seule fois la Belgique.

Explication d'une partie du programme:

J'ai écrit une class 'counter_double_key' avec une fonction 'count', qui permet de compter combien de personnes sont partis de leur pays, dans un autre. Par exemple 6 900 étudiants sont partis de France en Espagne, pour un semestre d'erasumus, 6 500 sont partis aux Royaumes-Unis. La fonction 'se place' dans un pays puis elle va passer dans le fichier csv et à chaque fois qu'elle rencontre un même pays de destination elle va faire plus 1. Si elle ne trouve plus le même pays, elle laisse le nombre déjà existent. Elle fera cette opérations jusqu'à avoir un dictionnaire avec tout les pays de l'EU en key et en value une liste avec toutes les destinations et le nombre de personnes par destinations.
