Rapport projet page Web

HTML:

Grace a HTML (Hyper Text Markup Language) il est possible de programmer un site web. HTML, est un document texte, qui utilise le markup language. Ce sont des tags diffrents avec differents utilites. Par exemple le tags 'strong' qui permet de mettre en gras une lettre, mais ce tag n'est pas le plus important. Les plus importants sont les tags tel que: 'html', 'head' et 'body', c'est un peu construit comme un corps humain. Dans le header vont etes ecrites toutes les informations les plus importantes pour la page web.(Ici vous trouvez tout les tags pour la "Tete".)

CSS:

Grace a CSS, il est possible d'embellir le site web ecrit avec HTML. Il est possible de faire plusieurs choses differentes, par exemple on peut changer la couleur de font ou de l'ecriture, on peut aussi changer le style de l'ecriture. Il est possible de faire les memes choses avec HTML, mais avec CSS on peut faire un style pour toutes les pages, c'est tres pratique!

Rapport projet créer une Carte

Base de données publique utilisée:

Pour ce projet il fallait créer une carte avec une base de données publique. J'ai trouvé mes données sur ce site internet. J'ai utlisé les tables cvs de 2011/2012 pour les étudiants et les professeurs en erasmus.

Démarche du projet:

En premier, j'ai analysé les fichiers csv afin de savoir quelles sont informations que j'ai besoin pour porjet. Ensuite j'ai commencé à écrire mon programme python, pour qu'il mette les informations nécessaire pour faire les graphique dans les charts.

lors de ce projet j'ai rencontré plusieurs problèmes. Premierement, c'était difficile de trouver un fichier qui est lisible. Ensuite, la Belgique a plusieurs 'countriecodes', alors il fallait les combiner afin d'avoir qu'une seule fois la Belgique.

Explication d'une partie du programme:

J'ai écrit une fonction 'count', qui permet de compter combien de personnes sont partis de leur pays, dans un autre. Par exemple 6 900 étudiants sont partis de France en Espagne, pour un semestre d'erasumus, 6 500 sont partis aux Royaumes-Unis. La fonction 'se place' dans un pays puis elle va passer dans le fichier csv et à chaque fois qu'elle rencontre le meme pays elle va faire plus 1. La totalité de mon programme est disponible ici
