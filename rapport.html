Rapport

HTML:

Grace a HTML (Hyper Text Markup Language) il est possible de programmer un site web. HTML, est un document texte, qui utilise le markup language. Ce sont des tags diffrents avec differents utilites. Par exemple le tags 'strong' qui permet de mettre en gras une lettre, mais ce tag n'est pas le plus important. Les plus importants sont les tags tels que: 'html', 'head' et 'body', c'est un peu construit comme un corps humain. Dans le header vont etes ecrites toutes les informations les plus importantes pour la page web.(Ici vous trouvez tout les tags pour la "Tete".)

CSS:

Grace a CSS, il est possible d'embellir le site web ecrit avec HTML. Il est possible de faire plusieurs choses differentes, par exemple on peut changer la couleur de font ou de l'ecriture, on peut aussi changer le style de l'ecriture. Il est possible de faire les memes choses avec HTML, mais avec CSS on peut faire un style pour toutes les pages, c'est tres pratique!
